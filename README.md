 Medical Insurance Cost Analysis

  Objective
- To analyze factors affecting insurance charges using data analytics.

  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

 Key Insights
- Smokers have significantly higher charges  
- BMI and age affect insurance cost  
- Data is right-skewed  
- Strong correlation between smoker and charges
  
  Conclusion
- Lifestyle factors like smoking have the highest impact on medical insurance costs.
